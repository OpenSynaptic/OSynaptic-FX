# Non-Arduino Precompiled Archives

These archives are retained for non-Arduino release channels and CI packaging.

Arduino tooling does not consume binaries from this folder.
For Arduino precompiled usage, place archives under:

- `src/{build.mcu}/libOSynapticFX.a`

Current non-Arduino artifacts:

- `osfx-v0.2.0-android.a`
- `osfx-v0.2.0-ios.a`
- `osfx-v0.2.0-linux.a`
- `osfx-v0.2.0-wasm.a`
